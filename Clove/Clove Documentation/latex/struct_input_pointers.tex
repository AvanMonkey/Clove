Struct to hold pointers and variables that we use in GLFW callback functions where it is not possible to pass by reference. Reason for multiple pointers is because \textquotesingle{}glfw\+Set\+Window\+User\+Pointer\textquotesingle{} only hold\textquotesingle{}s 1 pointer, so by storing all necessary pointers in a struct and pointing to this struct, we can access more than 1 pointer in callback functions.
